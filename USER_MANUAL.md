## 시작하기

### 시스템 요구사항

- **브라우저**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **모바일**: iOS Safari, Android Chrome
- **인터넷 연결**: 안정적인 인터넷 연결 권장
- **카메라**: QR 코드 스캔을 위한 카메라 권한 필요 (모바일)

### 앱 설치 (PWA)

1. 브라우저에서 도서관 예약 시스템 접속
2. 주소창의 설치 아이콘 클릭 또는 "앱 설치" 메뉴 선택
3. 설치 완료 후 바탕화면에서 앱 실행 가능

## 회원가입 및 로그인

### 회원가입

1. **메인 페이지 방문**: 도서관 예약 시스템 웹사이트 접속
2. **로그인 버튼 클릭**: 우측 상단 "로그인" 버튼 클릭
3. **회원가입 선택**: "계정 만들기" 또는 "회원가입" 링크 클릭
4. **정보 입력**:
   - 이메일 주소
   - 비밀번호 (8자 이상, 특수문자 포함 권장)
   - 이름 (선택사항)
5. **이메일 인증**: 가입한 이메일로 인증 메일 확인 및 인증 완료
6. **로그인**: 인증 완료 후 로그인하여 서비스 이용

### 로그인

1. **로그인 페이지 접속**: 우측 상단 "로그인" 버튼 클릭
2. **정보 입력**:
   - 이메일 주소
   - 비밀번호
3. **로그인 버튼 클릭**
4. **2단계 인증** (활성화된 경우): 인증 코드 입력

### 비밀번호 재설정

1. 로그인 페이지에서 "비밀번호를 잊으셨나요?" 링크 클릭
2. 등록된 이메일 주소 입력
3. 이메일로 전송된 재설정 링크 클릭
4. 새 비밀번호 설정

## 공간 검색 및 예약

### 공간 둘러보기

1. **메인 페이지 접속**: 로그인 후 메인 페이지로 이동
2. **공간 목록 확인**: 이용 가능한 모든 공간 표시
3. **공간 상세 정보**:
   - 공간 이름 및 유형
   - 최대 정원
   - 위치 정보
   - 상세 설명

### 검색 기능 사용

1. **검색창 활용**: 상단 검색창에 키워드 입력
   - 공간명 검색: "프로그래밍 워크숍"
   - 설명 검색: "초보자"
   - 위치 검색: "세미나실"
2. **실시간 필터링**: 입력과 동시에 결과 자동 필터링

### 필터 적용

1. **유형 필터**: 드롭다운에서 공간 유형 선택
   - 전체: 모든 유형 표시
   - 프로그램: 교육 프로그램
   - 강의실: 회의 및 강의용
   - 스터디룸: 개인/그룹 스터디용

2. **정원 필터**: 인원수에 따른 필터링
   - 전체: 모든 정원 표시
   - 소규모: 10명 이하
   - 중규모: 11-30명
   - 대규모: 30명 초과

3. **결과 확인**: 필터 적용 후 표시되는 공간 수 확인

### 예약하기

1. **공간 선택**: 관심 있는 공간 클릭
2. **예약 페이지 이동**: "예약하기" 버튼 클릭
3. **날짜 선택**: 달력에서 원하는 날짜 선택
4. **시간 선택**: 이용 가능한 시간대 선택
   - 시작 시간과 종료 시간 설정
   - 최소 30분, 최대 4시간 예약 가능
5. **예약 정보 확인**:
   - 선택한 공간 및 시간
   - 이용 인원 (선택사항)
   - 특별 요청사항 (선택사항)
6. **예약 확정**: "예약하기" 버튼 클릭

### 대기열 등록

공간이 만석인 경우:
1. **대기열 등록**: "대기열에 등록" 버튼 클릭
2. **알림 수신 대기**: 예약 취소 시 자동 알림 수신
3. **승급 확인**: 승급되면 이메일/SMS/푸시 알림 수신

## 예약 관리

### 내 예약 조회

1. **내 예약 페이지 접속**: 상단 메뉴 "내 예약" 클릭
2. **예약 목록 확인**: 모든 예약 내역 표시
3. **상태별 필터링**:
   - 전체: 모든 예약
   - 승인됨: 사용 가능한 예약
   - 대기중: 승인 대기중
   - 완료됨: 사용 완료

### 예약 수정

승인된 예약의 경우:
1. **수정 버튼 클릭**: 예약 카드의 "수정" 버튼 클릭
2. **시간 변경**: 새 시작/종료 시간 입력
3. **수정 확정**: "저장" 버튼 클릭
4. **승인 대기**: 관리자 재승인 필요

### 예약 취소

1. **취소 버튼 클릭**: 예약 카드의 "취소" 버튼 클릭
2. **확인 대화상자**: 취소 확인
3. **취소 완료**: 예약 상태 "취소됨"으로 변경
4. **대기자 승급**: 취소 시 대기자 자동 승급

## 체크인

### QR 코드 준비

1. **예약 확인**: 예약 당일 내 예약 페이지 확인
2. **QR 코드 표시**: 승인된 예약에 QR 코드 표시
3. **코드 저장**: 스크린샷 또는 이미지 저장 (선택사항)

### 체크인 방법

1. **체크인 페이지 접속**: 상단 메뉴 "체크인" 클릭
2. **카메라 권한 허용**: 브라우저 카메라 권한 요청 허용
3. **QR 코드 스캔**: 카메라로 QR 코드 스캔
4. **체크인 완료**: 성공 메시지 표시
5. **피드백 요청**: 이용 후 만족도 조사 참여 (선택사항)

### 체크인 시간

- **권장 시간**: 예약 시작 30분 전 ~ 시작 시간
- **최소 시간**: 예약 시작 10분 전까지
- **지각 체크인**: 시작 시간 10분 초과 시 노쇼 처리 가능

## 캘린더 뷰

### 캘린더 사용

1. **캘린더 페이지 접속**: 상단 메뉴 "캘린더" 클릭
2. **날짜 선택**: 달력에서 확인할 날짜 클릭
3. **예약 현황 확인**: 선택한 날짜의 모든 예약 표시
4. **상세 정보**: 각 예약의 시간, 공간, 정원 정보

### 예약된 날짜 표시

- **파란색 점**: 예약이 있는 날짜 표시
- **실시간 업데이트**: 새 예약 시 자동 반영

## 알림 설정

### 푸시 알림

1. **브라우저 권한**: 첫 방문 시 알림 권한 요청 허용
2. **알림 유형**:
   - 예약 승인/반려 알림
   - 체크인 리마인더 (시작 30분 전)
   - 대기열 승급 알림

### 알림 설정 변경

현재 버전에서는 브라우저 설정을 통해 알림을 제어할 수 있습니다:
1. 브라우저 설정 → 개인정보 및 보안 → 사이트 설정
2. 알림 권한 변경

## 문제 해결

### 자주 발생하는 문제

#### 로그인 문제

**문제**: 로그인이 되지 않음
**해결**:
1. 이메일 주소와 비밀번호 확인
2. 이메일 인증 완료 여부 확인
3. 브라우저 쿠키/캐시 삭제 후 재시도
4. 다른 브라우저에서 시도

#### 예약 실패

**문제**: 예약이 실패함
**해결**:
1. 선택한 시간대가 이미 예약되었는지 확인
2. 인터넷 연결 상태 확인
3. 페이지 새로고침 후 재시도

#### QR 코드 스캔 실패

**문제**: 체크인 시 QR 코드 인식 실패
**해결**:
1. 카메라 권한 허용 확인
2. 충분한 조명 확보
3. QR 코드가 화면에 완전히 표시되는지 확인
4. 스마트폰의 경우 줌 기능 해제

#### 알림이 오지 않음

**문제**: 푸시 알림 수신 실패
**해결**:
1. 브라우저 알림 권한 확인
2. 앱이 백그라운드에서 실행 중인지 확인
3. 브라우저 설정에서 알림 차단 해제

### 고객 지원

문제가 지속되는 경우:

1. **관리자에게 문의**: 시스템 내 문의 기능 사용
2. **이메일 지원**: support@library-system.com
3. **전화 지원**: 02-123-4567 (운영시간: 평일 9:00-18:00)

### 시스템 상태 확인

- **시스템 상태 페이지**: status.library-system.com
- **공지사항**: 메인 페이지 공지사항 확인
- **점검 일정**: 사전 공지 예정

## 이용 약관

### 서비스 이용 조건

1. **회원 자격**: 만 14세 이상 실명 인증 사용자
2. **예약 제한**: 1인당 최대 3건 동시 예약
3. **취소 정책**: 예약 시작 1시간 전까지 무료 취소
4. **노쇼 정책**: 3회 노쇼 시 1개월 이용 제한

### 책임 제한

- 시스템 장애로 인한 예약 불가 시 책임지지 않음
- 천재지변 등 불가항력 상황에 대한 책임 제한
- 사용자 부주의로 인한 손실에 대한 책임 제한
